# Impeccable Design Context

## Design Context

### Users
Solo quantitative trader using this as a personal desktop tool. Reviews weekly AI-generated technical analyses for Korean stocks (KOSPI/KOSDAQ). The context is focused work — reading through screening results, judging buy/hold/sell signals, drilling into detailed analysis. Not a casual browse; every session has a task. Desktop-only, single monitor assumed.

### Brand Personality
Warm, focused, personal. Feels like a well-organized personal trading journal — not a cold Bloomberg terminal, not a gamified retail app. The tool should feel like it was built specifically for this one person's workflow: opinionated, efficient, and quietly confident.

Three words: **warm · precise · yours**

Emotional goal: "I can see everything I need clearly, and it feels like mine."

### Aesthetic Direction
- **Theme**: Dark — already established with `slate-950` base + amber accent. Continue this direction. Refine with OKLCH for perceptual uniformity.
- **Color**: Near-black background with a warm amber undertone. Amber (`oklch(0.82 0.16 80)`) as the primary accent. Neutrals tinted warmly, not cold gray.
- **Typography**: Wanted Sans for headings, labels, and UI chrome. Pretendard Variable for body text, analysis content, and data. Both handle Korean+Latin gracefully.
- **Density**: Desktop-only allows for higher information density. Optimize for 1280px+ widths. Content areas can be wider; don't constrain everything to mobile-safe widths.
- **Vibe**: Dark editor, warm amber glow, crisp typography. Think: your own terminal that you actually enjoy looking at.

Anti-references: Generic dark dashboard with purple-to-blue gradients and neon accents. Anything that looks like a SaaS product for a thousand users.

### Design Principles
1. **Legibility first** — every decision is measured against how clearly the data reads. Dense is fine; unclear is not.
2. **Warm darkness** — the dark background should feel like a focused workspace, not a void. Amber warmth is the connective tissue throughout.
3. **Korean-native** — Korean text will appear throughout (stock names, analysis content). Typography and spacing must be calibrated for hangul, not retrofitted after the fact.
4. **Personal scale** — this tool serves one person. It can have opinions, skip defensive UX, and optimize for a single expert user's workflow.
5. **Quiet confidence** — no decorative noise. Every visual element should earn its presence. Restraint is a feature.
