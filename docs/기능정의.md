기능 정의 (최종)
1. 분석 실행 (CLI — 웹 UI 없음)
1-1. 실행 등록

새 실행을 생성하고 식별자를 발급받는다
메모(선택)를 함께 등록할 수 있다
1-2. 분석 결과 저장

종목 코드, 종목명, 분석 마크다운 결과, 사용 모델(Claude / GPT / Gemini)을 저장한다.
저장 시 마크다운에서 아래 항목을 자동 파싱하여 함께 저장한다
매매 판정 (매수 / 홀드 / 매도)
추세 (상승 / 하락 / 횡보)
구름대 위치 (구름 위 / 구름 안 / 구름 아래)
MA 배열 (정배열 / 역배열 / 혼조)
진입 가격, 1차 목표가, 손절 기준가
파싱 실패 시 어느 항목이 실패했는지 명시하여 오류를 반환한다. 저장은 거부한다.
수동 입력 (웹 UI)
2-1. 분석 결과 직접 등록

웹 UI에서 직접 분석 결과를 등록할 수 있다
입력 항목: 사용 모델(Claude / GPT / Gemini 선택), 분석 마크다운 붙여넣기
종목 코드와 종목명은 붙여넣은 마크다운에서 자동 파싱하여 추출한다
등록할 실행을 선택하거나, 메모(선택)와 함께 새 실행을 생성하여 등록할 수 있다
저장 시 CLI 경로와 동일하게 파싱을 수행하며, 실패 시 어느 항목이 실패했는지 화면에 표시하고 저장을 거부한다
조회 (웹 UI)
3-1. 분석 목록 페이지

분석 목록을 최신순으로 보여준다
각 항목에서 확인 가능한 정보: 실행 날짜, 분석 종목 수, 메모
3-2. 종목 목록 페이지

특정 실행에 속한 종목 목록을 보여준다
각 항목에서 확인 가능한 정보: 종목 코드, 종목명, 매매 판정, 사용 모델
매매 판정(매수/홀드/매도)으로 필터링할 수 있다
3-3. 분석 결과 상세 페이지

특정 종목의 분석 마크다운 전문을 렌더링하여 보여준다
파싱된 핵심 지표(매매 판정, 추세, 구름대, MA 배열, 가격대)를 요약 형태로 상단에 표시한다
해당 종목의 과거 실행 이력을 날짜 목록으로 보여준다 (다른 실행 시점 결과로 이동 가능)
